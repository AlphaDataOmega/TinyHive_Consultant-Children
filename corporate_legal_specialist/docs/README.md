## Contract Review Process

### Approval Thresholds
| Value | Approver |
|-------|----------|
| < $25K | Manager |
| $25K-100K | Director |
| $100K-500K | VP |
| > $500K | Legal + Finance |

### Review Timeline
| Contract Type | Target SLA |
|---------------|------------|
| Standard NDA | 1 business day |
| Customer contract | 3-5 business days |
| Vendor agreement | 3-5 business days |
| Complex deal | 7-10 business days |

## Privacy Compliance

### GDPR Requirements
| Principle | Implementation |
|-----------|---------------|
| Lawful basis | Consent or legitimate interest |
| Data minimization | Collect only necessary data |
| Retention limits | Delete when no longer needed |
| Subject rights | Honor access, deletion requests |
| DPAs | Require from all processors |

### CCPA Rights
| Right | Response Time |
|-------|---------------|
| Know | 45 days |
| Delete | 45 days |
| Opt-out | Immediate |
| Non-discrimination | Ongoing |
