## Core Competency Areas

### 1. Financial Operations and Budgeting

#### Budget Structure
- Quarterly budget cycles aligned with OKRs
- Category-based spending organization
- Single owner per spending category
- Internal transparency of all budgets

#### Standard Budget Categories
| Category | Typical Owner |
|----------|---------------|
| Technical Infrastructure | Engineering Lead |
| Marketing | Marketing Lead |
| Business Development | BD Lead |
| Community Outreach | Community Manager |
| Security | Security Lead |
| Rent | Operations |
| Administration | Operations |
| Design Resources | Design Lead |
| Legal | Legal/Operations |

#### Budget Best Practices
- Set budgets immediately after OKR planning
- Exclude salaries for early-stage companies (first 10 employees)
- Require approval from category owner for all expenses
- Report actuals vs. budget at quarter end
- Maintain public visibility internally

### 2. Meeting Protocols

#### Core Hours Framework
- **Core meeting hours**: 10:30 AM - 1:00 PM (local timezone)
- **Standard notice**: 24+ hours for core hours meetings
- **Extended notice**: 1 week for outside core hours

#### Punctuality Standards
- Leaders start meetings on time
- Attendees arrive on time
- No grace periods

#### Remote Meeting Requirements
- Video link provided before meeting start
- Remote attendees join early
- Quiet location with reliable internet required
- Phone dial-ins from public places prohibited
- Voice-only at organizer discretion

#### Technical Prerequisites
- Fast, reliable internet connection
- Quiet environment
- Functional audio/video equipment
- Pre-tested setup

### 3. Innovation Programs (Hack Weeks)

#### Program Structure
- **Timing**: Sixth week of every quarter
- **Duration**: One week
- **Team size**: 1-4 people
- **Focus**: Company-related projects

#### Participation Guidelines
- Anyone can propose projects
- Self-selection into teams
- Cross-functional teams encouraged
- Solo projects acceptable

#### Week Operations
- Normal work pauses
- Group lunches (ordered or prepared together)
- Communal atmosphere
- High energy and collaboration

#### Optional Enhancements
- Off-site locations
- Guest speakers

#### Week Conclusion
- Friday: Project presentations
- Company celebration

### 4. Interview Process

#### Three-Stage Framework

**Stage 1: Informal/Coffee (30 minutes)**
- Format: Phone, video, in-person, or coffee shop
- Purpose: Mutual interest assessment
- Activities: Conversational Q&A, company sell
- Expected filter: ~80% of candidates

**Stage 2: Technical Interview (2 hours)**
- 45 min: High-level thinking exercise
- 45 min: Implementation exercise
- 30 min: Q&A session
- Expected filter: 50-70% of remaining

**Stage 3: Team Fit (Full Day)**
- Morning: Extended challenge exercise
- Lunch: Team lunch
- Afternoon: One-on-one meetings
- Requirements: On-site, travel covered

#### Pre-Interview Requirements
- Candidate work samples reviewed
- Company values shared
- Position information provided
- Self-screening opportunity given

### 5. File Sharing and Organization

#### Base Directory Structure
| Directory | Purpose | Access |
|-----------|---------|--------|
| Company Handbook | How-we-work docs | All, commentable |
| Active Projects | Current work | All |
| Inactive Projects | Archived work | All |
| Resources | Shared assets | All |
| OKRs | Objectives/Results | All |
| [Employee Name] | Personal docs | Individual + HR |

#### Project Lifecycle
- **Creation**: `Project Name` in Active Projects
- **Archive**: `YYYY-MM Project Name` in Inactive Projects

#### Scaling Considerations
- Structure optimized for autonomous projects
- Team-based organization needed as company grows
- Quarterly structure reviews recommended

### 6. Candidate Sourcing

#### Channel Overview

| Channel | Strengths | Considerations |
|---------|-----------|----------------|
| Referrals | Highest quality signal | Potential homogeneity bias |
| Outbound | Targeted approach | Time-intensive |
| Inbound | Broadest pool | Lower initial filter |

#### Referral Program
- Monthly recruiting lunch (second Tuesday)
- Network review sessions
- $5,000 typical referral bonus
- Diversity emphasis required
- Bias awareness training

#### Outbound Recruiting
- Position ownership assigned
- Proactive candidate searches
- Diverse pool targeting
- Partner organization engagement

#### Diversity Partners (Examples)
- Hack the Hood
- Women Who Code
- Qeyno Labs
- Local diversity-focused groups

#### Inbound Management
- All positions posted on company website
- Strategic external board posting
- 3-day response requirement
- Personal rejection before ATS update
