# Performance Management Specialist - Documentation

## Overview

This directory contains reference documentation for the Performance Management Specialist agent. The agent provides expert guidance on employee performance management processes including reviews, feedback, talent assessment, and career development.

## Primary Reference

- **SOP-HR-Performance-Management.md** — Comprehensive Standard Operating Procedure covering the full performance management lifecycle

## Key Topics Covered

### Performance Review Cycles
- New employee reviews (3, 6, 12 month milestones)
- Annual performance review process and timeline
- Mid-year check-ins and ongoing feedback

### Goal Setting and Individual Growth Plans
- IGP components: self-reflection, career direction, setting intentions, development goals
- The 70-20-10 development rule (on-the-job learning, interactions, formal training)
- Accomplishment tracking and progress monitoring

### 360-Degree Feedback
- When to use 360 feedback
- Process for team members and managers
- Standard feedback questions
- Eligibility requirements (6+ months tenure)

### Talent Assessment
- Performance/Growth Potential Matrix (9-Box model)
- Performance ratings: Exceeding, Performing, Developing
- Growth potential ratings and the four pillars
- Calibration process best practices
- Expected distribution guidelines

### Feedback Guidelines
- The SBI Model (Situation-Behavior-Impact)
- Guidelines for giving and receiving feedback
- Recognition best practices
- 4:1 positive to constructive feedback ratio

### Promotions and Career Development
- Promotion philosophy and types (in-cycle, internal mobility)
- Promotion process stages: planning, calibration, processing
- Documentation requirements (maximum 3 pages)
- Compensation guidelines (5-10% typical increase)
- Career development vs. mobility distinctions

### Performance Improvement Plans (PIPs)
- When PIPs apply (and when they do not)
- Underperformance definition (4 consecutive weeks)
- Process for addressing underperformance
- Goal setting during remediation

### Compensation Review Conversations
- Preparation requirements
- Communication best practices
- Sample scripts for increases and non-increases
- Handling team member reactions

## Roles and Responsibilities

| Role | Key Responsibilities |
|------|---------------------|
| **Employee** | Own development, complete self-assessments, maintain accomplishment tracking |
| **Manager** | Facilitate career conversations, provide feedback, complete assessments, address concerns |
| **HR/People Team** | Provide tools and training, facilitate calibration, process changes |
| **Leadership** | Review promotions, participate in calibration, champion feedback culture |

The Performance Management Specialist can assist with:
- Designing performance review cycles for your organization
- Creating 360-degree feedback programs
- Building Individual Growth Plan templates
