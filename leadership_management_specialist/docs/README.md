## Manager of One Philosophy

### Core Principles
| Principle | Description |
|-----------|-------------|
| Self-direction | Set your own priorities and goals |
| Accountability | Own outcomes, not just tasks |
| Proactivity | Act without being asked |
| Documentation | Write things down as they happen |
| Efficiency | Avoid unnecessary meetings |

## 1-on-1 Meeting Framework

### Meeting Structure
| Component | Duration | Focus |
|-----------|----------|-------|
| Personal check-in | 5 min | Well-being, rapport |
| Team member topics | 20 min | Their agenda items |
| Manager topics | 10 min | Updates, feedback |
| Career development | 10 min | Growth, goals |
| Action items | 5 min | Next steps |

### Frequency Guidelines
| Team Tenure | Frequency |
|-------------|-----------|
| < 3 months | Weekly |
| 3-12 months | Bi-weekly |
| > 12 months | Bi-weekly or monthly |

## Feedback Framework

### ASK Model
| Element | Question |
|---------|----------|
| Actionable | What can they do differently? |
| Specific | What exact behavior/example? |
| Kind | Does this help them improve? |

## GROW Coaching Model

| Phase | Focus | Questions |
|-------|-------|-----------|
| Goal | Desired outcome | What do you want to achieve? |
| Reality | Current situation | Where are you now? |
| Options | Possibilities | What could you do? |
| Will | Commitment | What will you do? |

## Emotional Intelligence

### Five Components
| Component | Description |
|-----------|-------------|
| Self-awareness | Know your emotions |
| Self-regulation | Control reactions |
| Motivation | Internal drive |
| Empathy | Understand others |
| Social skills | Manage relationships |

## Conflict Resolution Styles

| Style | When to Use |
|-------|-------------|
| Collaborating | Important issues, time available |
| Competing | Quick decisions, unpopular actions |
| Compromising | Moderate importance, time pressure |
| Avoiding | Trivial issues, emotions high |
| Accommodating | Issue more important to other party |

## Underperformance Process

| Phase | Actions |
|-------|---------|
| Identify | Document specific gaps |
| Diagnose | Skills vs. will issue |
| Communicate | Clear expectations, timeline |
| Support | Resources, coaching, training |
| Evaluate | Track progress |
| Decide | Continue, reassign, or exit |

## Trust Building (Remote Teams)

| Dimension | Actions |
|-----------|---------|
| Reliability | Follow through on commitments |
| Transparency | Share context openly |
| Competence | Demonstrate expertise |
| Inclusion | Involve others in decisions |
| Psychological safety | Make it safe to fail |
