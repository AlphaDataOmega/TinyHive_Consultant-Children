# Civil Litigation Specialist Documentation

## Overview

This directory contains reference documentation for the Civil Litigation Specialist agent, supporting federal court civil litigation procedures and FRCP compliance.

## Scope

The Civil Litigation Specialist provides guidance on:

- **Summons (Form AO 440)** — Official notification to defendants of civil lawsuits filed in federal court
- **Subpoenas (Form AO 88 Series)** — Court orders for testimony (AO 88A) or document production (AO 88B)
- **Judgment (Form AO 450)** — Official court records documenting final case disposition

## Response Timeframes

| Defendant Type | Response Deadline |
|----------------|-------------------|
| Individual defendants | 21 days from service |
| US Government | 60 days from service |

## Filing Methods

### Electronic Filing (CM/ECF)
1. Log into the court's CM/ECF system
2. Select appropriate event category
3. Upload completed PDF form
4. Verify filing confirmation receipt

### Paper Filing
1. Prepare original and required copies
2. Include filing fee or fee waiver if applicable
3. Obtain file-stamped copies for records
4. Note docket entry confirmation

## Record Retention

| Document Type | Retention Period |
|---------------|------------------|
| Summons (filed copy) | Duration of case + 7 years |
| Subpoenas (issued) | Duration of case + 3 years |
| Judgment (certified copy) | Permanent |
| Proof of service | Duration of case + 7 years |
