## M&A Process Stages

| Stage | Duration | Key Activities |
|-------|----------|----------------|
| Exploratory | 2-4 weeks | Initial contact, NDA, product calls |
| Early Diligence | 4-6 weeks | Technical review, integration plan |
| Confirmatory | 2-4 weeks | Full diligence, definitive docs |
| Close | 1-2 weeks | Signing, Day 1 prep |
| Integration | 3-12 months | Full integration execution |

## Due Diligence Workstreams

| Workstream | Focus Areas |
|------------|-------------|
| Technical | Code quality, architecture, tech debt |
| Product | Roadmap, customer value, integration |
| Financial | Revenue, costs, projections |
| Legal | IP, contracts, liabilities |
| People | Team, culture, retention risk |
| Security | Vulnerabilities, compliance |
