## RICE Prioritization

| Factor | Description | Scale |
|--------|-------------|-------|
| Reach | Users affected | Number |
| Impact | Value delivered | 0.25-3 |
| Confidence | Certainty level | % |
| Effort | Person-months | Estimate |

### RICE Score
```
Score = (Reach × Impact × Confidence) / Effort
```
