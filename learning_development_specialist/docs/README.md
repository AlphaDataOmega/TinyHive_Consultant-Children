# Learning Development Specialist Documentation

## Overview

This directory contains reference materials and procedures for the Learning Development Specialist consultant agent. The specialist provides expert guidance on learning programs, talent development, mentorship, career growth, and compliance training.

## Core Competencies

### Learning Programs
- Blended learning design (ILT, VILT, e-learning, workshops)
- Learning Management System (LevelUp) administration
- Content partner integrations (O'Reilly, Hone, Masterclass, Google Cloud Skills Boost)
- SmartCards, Pathways, and Journeys content creation
- Adult learning theory application

### Talent Development
- 70-20-10 development model implementation
- Individual Growth Plan (IGP) facilitation
- Accomplishment and health tracker management
- Skill-based learning pathways
- Cross-functional internship programs

### Growth and Development Fund
- Eligibility assessment and application guidance
- Tiered approval workflow management:
  - $0 - $1,000: Manager approval
  - $1,000.01 - $5,000: Manager + VP approval
  - $5,000.01 - $10,000: Manager + Executive approval
- Eligible program categories:
  - Academic study
  - Professional certifications
  - Conferences and workshops
  - Self-service learning subscriptions
  - Language courses
  - Code-based learning
  - Professional coaching

### Career Development
- Career conversation facilitation
- Career development vs career mobility guidance
- Inclusive growth consideration frameworks
- Promotion preparation support
- Job family and framework alignment

### Mentorship Programs
- Mentor-mentee matching
- Goal-setting frameworks (time-bound, specific, measurable, personal)
- Meeting structure and cadence recommendations
- Program duration guidelines (3-6 months typical)
- Mentorship closure and documentation

### Manager Development
- Elevate program tiers:
  - Elevate: Managers and Senior Managers
  - Elevate Applied: Continuous learning
  - Elevate+: Directors and Senior Directors
  - ElevatePrep: High-potential ICs
- New manager transition support
- Leadership competency development

### Compliance Training
- New hire training enrollment
- Annual refresher management
- Notification cadence administration
- Escalation procedures for overdue training
- Security awareness and secure coding training

## Success Metrics

| Program | Key Metrics |
|---------|-------------|
| Elevate | 85% completion rate, NPS scores, behavioral change |
| Mentorship | 80+ NPS, 100% call participation, goal achievement |
| Learning Platform | Completion rates, compliance timeliness, engagement |
