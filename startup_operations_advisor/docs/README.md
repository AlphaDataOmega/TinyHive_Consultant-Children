# Startup Operations Advisor Documentation

## Overview

This documentation supports the `startup_operations_advisor` consultant child, providing reference materials and guidelines for startup company operations and policies.

## Core Competency Areas

### 1. Company Values and Culture
- Core values framework (Communication, Results, Iteration)
- Remote-first culture advantages
- Open company principles
- Transparency and documentation practices

### 2. Communication Guidelines
- Asynchronous communication first approach
- Effective communication principles (assume positive intent, kindness, feedback)
- Sources of truth hierarchy (primary vs ephemeral)
- Meeting guidelines (internal and external)
- Video call and Slack best practices
- Email communication protocols

### 3. Time Off and Leave Policies
- Flexible time off philosophy
- Paid time off (PTO) policy (minimum 30 days)
- Permissionless time off process
- Parental leave (12 weeks paid)
- Emergency and sick leave

### 4. Spending and Expense Policies
- Core philosophy: spend company money like your own
- Expense categories (trivial vs larger purchases)
- Equipment policy (laptops, monitors, accessories)
- Software expenses and collaborative stack preferences

### 5. Security Policies
- Security requirements and compliance
- Password management (password manager required)
- Data protection and encryption
- Billing security (PCI compliance)
- Incident response procedures

### 6. Privacy Policies
- Core privacy principles (your data is your data)
- Data collection categories
- User data rights (GDPR/CCPA compliance)
- Data retention and deletion timelines

### 7. Terms of Service Framework
- Account terms
- Payment and cancellation terms
- Modifications and uptime guarantees
- Copyright and liability provisions

### 8. Service Level Agreements
- Uptime guarantee (99.99% for premium tiers)
- Credit structure for outages
- Monitoring and status pages

### 9. Engineering Management
- One-on-one meeting best practices
- Performance review components
- Manager change transitions
- Role transitions (IC to Manager and vice versa)

### 10. Remote Work Best Practices
- Hybrid call solutions (everyone uses own equipment)
- Calendar management and sharing
- Productivity recommendations
- Essential and recommended equipment

## Quick Reference

### Communication
- Async first, sync when needed (after 3 back-and-forths)
- Write things down in sources of truth
- Use public channels over DMs
- Start and end meetings on time

### Time Off
- Minimum 30 days PTO per year
- 5 days notice for planned leave
- 10+ days requires additional approval
- Ensure coverage before leaving

### Expenses
- Under $75: Just buy it
- Keep all receipts
- Spend as if it were your own money
- Document new expense categories

### Security
- Use password manager
- Strong unique passwords
- Report incidents immediately
- Follow data protection guidelines

### Remote Work
- Everyone uses own equipment for calls
- Keep calendar up to date
- Communicate availability
- Respect timezone differences
