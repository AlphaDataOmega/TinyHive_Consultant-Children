## Core Knowledge Domains

### Onboarding Timeline
| Phase | Duration | Focus |
|-------|----------|-------|
| Pre-boarding | Before day 1 | Setup, welcome email |
| Day 1 | First day | Orientation, buddy intro |
| Week 1 | Days 1-5 | Team integration, tools |
| Month 1 | Days 1-30 | Role clarity, first deliverables |
| Month 3 | Days 1-90 | Full productivity |

### Buddy Program
- Assigned on day one
- Informal support and guidance
- Regular check-ins
- Cultural integration help

### Time Off Types
| Type | Description |
|------|-------------|
| Flexible PTO | Self-managed time off |
| Sick Time | Illness or medical |
| Parental Leave | New parents |
| Bereavement | Family loss |
| Military | Service obligations |
| Jury Duty | Civic duty |

### Competency Framework Levels
1. **Developing** - Learning the role
2. **Performing** - Meeting expectations
3. **Exceeding** - Above expectations
4. **Expert** - Domain authority

### 360 Feedback Process
1. Self-assessment
2. Manager feedback
3. Peer feedback
4. Direct report feedback (if applicable)
5. Synthesis and action planning

### People Ops SLAs
| Request Type | Response Time |
|--------------|---------------|
| Urgent | Same day |
| Standard | 3 business days |
| Complex | 5 business days |
