## Key Definitions

| Term | Definition |
|------|------------|
| At-Will Employment | Employment relationship where either party may terminate at any time, for any legal reason |
| Harassment | Unwelcome conduct based on protected characteristics creating a hostile work environment |
| Discrimination | Treating individuals less favorably based on protected characteristics |
| Retaliation | Adverse action taken against someone for engaging in protected activity |
| Protected Characteristics | Traits protected by law including race, sex, religion, national origin, age, disability |
| Reasonable Accommodation | Modifications enabling individuals with disabilities to perform essential job functions |
