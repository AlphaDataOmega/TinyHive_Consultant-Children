# Compensation Specialist Documentation

## Overview

This directory contains reference materials for the Compensation Specialist consultant, focusing on compensation strategy, equity programs, and total rewards administration.

## Core Competencies

### Compensation Philosophy
- Market competitiveness targeting 50th percentile or above
- Pay for performance alignment
- Transparency through standardized frameworks
- Location-based pay using cost of labor analysis
- Internal equity ensuring consistent pay for same attributes

### Compensation Elements

| Component | Description |
|-----------|-------------|
| Base Salary | Annual guaranteed pay determined by role, level, and location |
| Variable Pay | Performance-based bonuses or commissions (OTE for sales) |
| Equity | RSUs or stock options representing ownership |
| Benefits | Healthcare, retirement, PTO, professional development |

### Compensation Calculator Formula

```
Compensation = SF Benchmark x Location Factor x Level Factor x Exchange Rate
```

**Output**: Base + Variable = Total Target Cash (TTC)

### Level Factors

| Level | Factor |
|-------|--------|
| Junior | 0.8 x ic_ttc |
| Intermediate | 1.0 x ic_ttc |
| Senior | 1.2 x ic_ttc |
| Staff/Manager | 1.0 x manager_ttc |
| Senior Manager | 1.2 x manager_ttc |
| Director | 1.0 x director_ttc |
| Senior Director | 1.0 x seniordirector_ttc |

## Equity Compensation

### Types
- **ISOs (Incentive Stock Options)**: US employees with special tax treatment
- **NQs (Non-Qualified Options)**: Contractors and non-US employees
- **RSUs (Restricted Stock Units)**: Grants of company common stock

### Standard Vesting Terms
- Four-year vesting schedule
- One-year cliff for new hire grants
- Monthly vesting after cliff
- 90 days post-termination to exercise vested options

### Grant Types
- **New Hire Grants**: Part of offer packages with 6-month cliff
- **Promotion Grants**: Differential between new and previous level
- **Refresh Grants**: Annual review based on talent assessment

## Annual Compensation Review (ACR)

### Merit Guidelines

| Performance Rating | Recommended Increase |
|-------------------|---------------------|
| Developing | 0% |
| Performing | 2% to 5% |
| Exceeding | 4% to 8% |
| Too New To Rate | 0% to 4% |

### Equity Refresh Guidelines

| Growth Potential | Target Range |
|-----------------|--------------|
| Developing | Not funded |
| Growing | Minimum to midpoint |
| Exceeding | Midpoint to maximum |
| Key Talent | Maximum to 2x maximum |

## Incentive Programs

### Discretionary Bonuses
- $1,000 USD fixed amount for peer-to-peer recognition
- Must be outside normal role scope with exceptional execution

### Working Group Bonus
- $100 per person for group achievements

### Company Awards
- Values Awards: $5,000 USD per award
- Special Innovation Awards: $10,000 USD

## Key Metrics

### Compa-Ratio Interpretation

| Compa-Ratio | Position |
|-------------|----------|
| Below 0.80 | Below targeted market position |
| 0.80 to 1.00 | Between minimum and midpoint |
| 1.00 to 1.20 | Between midpoint and maximum |
| Above 1.20 | Above targeted market position |
